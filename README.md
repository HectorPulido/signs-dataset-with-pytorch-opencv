# SIGNS dataset image detector with Pytorch and Opencv
This is a image detector made with pytorch, also contains an opencv interface for inference.

<br>

![Example](demo.gif)

## How to train
1. Install pytorch, numpy, matplotlib
2. Execute the "train.ipynb" notebook
3. Check the "model" file saved in the last cell of "train.ipynb"

## How to infer
1. Install pytorch, OpenCv
2. Execute the script "app_predict.py"
3. Make sure to have a white background to get the correct infer
